# FX 過去検証トレーナー v2

FXトレード学習用の過去検証ツール。**手動検証**(裁量トレード練習)と**自動検証**(ストラテジーの自動バックテスト)の2モードを搭載。

インストール・ビルド不要。`index.html` をブラウザで開くだけで動作します。

## 使い方

### 🎮 手動検証モード
1. 「次の足へ」(`Space`)でチャートを1本ずつ進める
2. 「買い」(`B`)/「売り」(`S`)でエントリー、SL/TPはpips指定
3. 成績(勝率・RR・最大DDなど)と損益カーブがリアルタイム更新

### ⚙️ 自動検証モード
1. 戦略を選択(MAクロス / RSI逆張り / ブレイクアウト)
2. パラメータ・SL/TP・スプレッドを設定
3. 「バックテスト実行」→ 全データ(1500本)に自動適用
4. チャート上に全売買マーカー(▲▼=エントリー、×=決済)、成績・履歴・損益カーブを表示
5. **シード値**を指定すると同じチャートを再現可能 → 同一データで戦略やパラメータを比較検証できます

## Claude Code での開発

このプロジェクトには `CLAUDE.md`(アーキテクチャ・規約・ロードマップ)が含まれており、Claude Code がすぐに作業を始められます。

```bash
cd fx-backtester
claude
```

例えばこんな指示から始められます:
- 「CLAUDE.mdのPhase 2にあるCSVインポート機能を実装して」
- 「ボリンジャーバンド逆張り戦略を追加して」
- 「パラメータのグリッドサーチ最適化を作って」

## ファイル構成

```
fx-backtester/
├── index.html        # メインUI
├── css/style.css     # ダークテーマ
├── js/
│   ├── data.js       # 価格データ生成(シード付き・再現可能)
│   ├── strategies.js # 売買戦略の定義
│   ├── engine.js     # バックテストエンジン・統計
│   ├── chart.js      # Canvasチャート描画
│   └── app.js        # UI制御
├── CLAUDE.md         # Claude Code用の開発ガイド
└── README.md
```
